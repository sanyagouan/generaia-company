# Contribuir a GeneraIA Company

Este repo es gestionado por **Paperclip CEO** con apoyo de **Hermes Agent**.

## Flujo de trabajo

1. Paperclip CEO crea/modifica documentos en este repo
2. Hermes Agent valida y optimiza
3. Revisión final antes de merge a main

## Documentos (PAPs)

- `docs/PAP-*.md` — Documentos del proyecto actual
- `skills/` — Skills operativos del agente

## Nomenclatura de commits

```
feat: nueva funcionalidad
doc: actualización documental
fix: corrección
refactor: mejora sin cambio funcional
```
